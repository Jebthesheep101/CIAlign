Installation¶

Requirements¶

  • python >= 3.6

  • matplotlib >= 2.1.1

  • numpy >= 1.16.3

  • scipy >= 1.3.0

Recommendations¶

The easiest way to install CIAlign is using conda or pip3.

Conda¶

conda install -c bioconda cialign

link

pip3¶

pip3 install cialign

link
